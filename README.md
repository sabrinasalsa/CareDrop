# CareDrop

Sistem Layanan Donasi Barang berbasis Web

## Deskripsi
Website ini akan menyediakan fitur publikasi kebutuhan yayasan/posko dan penawaran donasi barang, sehingga donatur dapat dengan mudah menemukan dan menyalurkan bantuan kepada posko atau yayasan yang tepat. Dengan adanya platform ini, diharapkan alur penyaluran donasi menjadi lebih transparan, terstruktur, dan tepat sasaran.

## Alamat Website
Localhost

## Menu Utama

```text
USER (DONATUR)
  - Landing Page (Katalog Kebutuhan)
  - Sign Up
  - Login
    - Mengajukan Tawaran Donasi Barang
    - Memasukkan Nomor Resi Pengiriman
  - Melacak Status Pengiriman Barang (Timeline)
  - Melihat Riwayat Donasi

YAYASAN / POSKO
  - Landing Page
  - Sign Up
  - Login
    - Menambahkan/Menutup Daftar Kebutuhan Barang
    - Menyetujui atau Menolak Tawaran Donasi
    - Mengonfirmasi Penerimaan Barang Fisik
  - Melihat Laporan Penerimaan Donasi
  - Mengelola Profil dan Berkas Legalitas

ADMIN
  - Landing Page
  - Login
    - Verifikasi Pendaftaran Yayasan Baru
    - Mengelola Data Kategori Barang (Master Data)
    - Memantau Dasbor Analitik & Statistik

```
## Teknologi
